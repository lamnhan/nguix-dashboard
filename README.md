## Getting started

- Install:

```sh

```

- Usage:

Create the module `app-dashboard.module.ts`, see [example](https://github.com/lamnhan/nguix-dashboard/blob/main/src/app/app-dashboard.module.ts).

For more, visit: <https://nguix-dashboard.lamnhan.com>
